# 02 — Transformasi Fourier untuk Sinyal Diskrit

## 2.1 Motivasi

Analisis Fourier adalah alat terpenting dalam DSP. Ide dasarnya: **setiap sinyal dapat direpresentasikan sebagai superposisi gelombang sinus/cosinus dengan frekuensi berbeda**. Ini memungkinkan kita memahami sinyal dalam domain **frekuensi**, bukan hanya domain **waktu**.

---

## 2.2 Discrete-Time Fourier Transform (DTFT)

DTFT mentransformasi sinyal diskrit $x[n]$ ke representasi frekuensi kontinu:

$$X(e^{j\omega}) = \sum_{n=-\infty}^{\infty} x[n] e^{-j\omega n}$$

### Inverse DTFT

$$x[n] = \frac{1}{2\pi} \int_{-\pi}^{\pi} X(e^{j\omega}) e^{j\omega n} d\omega$$

### Properti DTFT

| Properti | Domain Waktu | Domain Frekuensi |
|----------|-------------|-----------------|
| Linearitas | $ax_1[n] + bx_2[n]$ | $aX_1(e^{j\omega}) + bX_2(e^{j\omega})$ |
| Time Shift | $x[n-k]$ | $e^{-j\omega k} X(e^{j\omega})$ |
| Frequency Shift | $e^{j\omega_0 n} x[n]$ | $X(e^{j(\omega-\omega_0)})$ |
| Konvolusi | $x_1[n] * x_2[n]$ | $X_1(e^{j\omega}) \cdot X_2(e^{j\omega})$ |
| Perkalian | $x_1[n] \cdot x_2[n]$ | $\frac{1}{2\pi} X_1 \circledast X_2$ |
| Konjugat | $x^*[n]$ | $X^*(e^{-j\omega})$ |
| Parseval | $\sum \|x[n]\|^2$ | $\frac{1}{2\pi} \int_{-\pi}^{\pi} \|X(e^{j\omega})\|^2 d\omega$ |

### DTFT Sinyal-Sinyal Dasar

| Sinyal $x[n]$ | DTFT $X(e^{j\omega})$ |
|---------------|----------------------|
| $\delta[n]$ | $1$ |
| $\delta[n-k]$ | $e^{-j\omega k}$ |
| $u[n]$ | $\frac{1}{1-e^{-j\omega}} + \pi\delta(\omega)$ |
| $a^n u[n]$, $\|a\|<1$ | $\frac{1}{1-ae^{-j\omega}}$ |
| $1$ | $2\pi\delta(\omega)$ |
| $e^{j\omega_0 n}$ | $2\pi\delta(\omega-\omega_0)$ |

## 2.3 Discrete Fourier Transform (DFT)

DFT menghitung representasi frekuensi **diskrit** dari sinyal diskrit sepanjang $N$ sampel:

$$X[k] = \sum_{n=0}^{N-1} x[n] e^{-j\frac{2\pi}{N}kn}, \quad k = 0, 1, ..., N-1$$

### Inverse DFT (IDFT)

$$x[n] = \frac{1}{N} \sum_{k=0}^{N-1} X[k] e^{j\frac{2\pi}{N}kn}, \quad n = 0, 1, ..., N-1$$

### Twiddle Factor

$$W_N = e^{-j\frac{2\pi}{N}}$$

Sehingga: $X[k] = \sum_{n=0}^{N-1} x[n] W_N^{kn}$

### Interpretasi Bin Frekuensi

Bin ke-$k$ dalam DFT merepresentasikan frekuensi analog:
$$f_k = \frac{k \cdot f_s}{N}$$

di mana $f_s$ adalah frekuensi sampling.

**Resolusi frekuensi**: $\Delta f = \frac{f_s}{N}$

Untuk meningkatkan resolusi: **perbesar N** (lebih banyak sampel).

### Zero-Padding

Menambahkan nol di akhir sinyal sebelum DFT → **interpolasi** spektrum (bukan meningkatkan informasi), memberikan spektrum yang lebih halus secara visual.

### Kompleksitas Komputasi DFT Naif

$$\text{Kompleksitas} = O(N^2) \text{ (perkalian kompleks)}$$

Untuk $N = 1024$: ~1 juta operasi!

---

## 2.4 Fast Fourier Transform (FFT)

FFT adalah algoritma efisien untuk menghitung DFT. Algoritma paling populer: **Cooley-Tukey Radix-2 Decimation-in-Time (DIT)**.

### Kompleksitas FFT

$$\text{Kompleksitas} = O(N \log_2 N)$$

Untuk $N = 1024$: ~10.000 operasi — **100× lebih cepat** dari DFT naif!

### Ide Dasar Cooley-Tukey

DFT panjang $N$ dipecah menjadi dua DFT panjang $N/2$:

$$X[k] = \underbrace{\sum_{n=0}^{N/2-1} x[2n] W_N^{2kn}}_{X_\text{even}[k]} + W_N^k \underbrace{\sum_{n=0}^{N/2-1} x[2n+1] W_N^{2kn}}_{X_\text{odd}[k]}$$

Karena $W_N^2 = W_{N/2}$:

$$X[k] = X_\text{even}[k] + W_N^k \cdot X_\text{odd}[k]$$
$$X[k+N/2] = X_\text{even}[k] - W_N^k \cdot X_\text{odd}[k]$$

Inilah **butterfly operation** — operasi dasar FFT.

### Butterfly Diagram

```
a ──┬──────────────── a + W*b
    │        ↑
    └──×W───┤
           ↓
b ──────────────────── a - W*b
```

## 2.5 Properti Simetri DFT

Untuk sinyal **real-valued** $x[n]$:
- $X[N-k] = X^*[k]$ (simetri Hermitian)
- Cukup menyimpan $N/2 + 1$ nilai DFT
- $|X[k]| = |X[N-k]|$ (simetri magnitude)
- $\angle X[k] = -\angle X[N-k]$ (antisimetri fase)

---

## 2.6 Short-Time Fourier Transform (STFT)

DTFT/DFT mengasumsikan sinyal **stasioner** (spektrum tidak berubah terhadap waktu). Untuk sinyal **non-stasioner** (suara, musik, EEG), kita perlu representasi **waktu-frekuensi**.

### Definisi STFT

$$X[n, k] = \sum_{m=-\infty}^{\infty} x[m] \cdot w[m-n] \cdot e^{-j\frac{2\pi}{N}km}$$

di mana $w[n]$ adalah **window function** yang mengekstrak segmen lokal sinyal di sekitar waktu $n$.

### Spectrogram

$$\text{Spectrogram}[n, k] = |X[n, k]|^2$$

### Tradeoff Resolusi (Prinsip Ketidakpastian Heisenberg)

$$\Delta t \cdot \Delta f \geq \frac{1}{4\pi}$$

- **Window pendek**: resolusi waktu baik, resolusi frekuensi buruk
- **Window panjang**: resolusi frekuensi baik, resolusi waktu buruk

### Window Functions Populer

| Window | Karakteristik | Penggunaan |
|--------|--------------|-----------|
| **Rectangular** | Resolusi frekuensi terbaik, sidelobe tinggi | Sinyal stasioner |
| **Hanning** | Sidelobe rendah, resolusi sedang | Penggunaan umum |
| **Hamming** | Sidelobe sangat rendah | Audio |
| **Blackman** | Sidelobe sangat rendah, lebar mainlobe besar | Pengukuran presisi |
| **Kaiser** | Parameter $\beta$ adjustable | Desain filter |

---

## 2.7 Circular vs Linear Convolution

DFT mengasumsikan sinyal **periodik** → konvolusi melalui DFT adalah **konvolusi sirkular**, bukan linear!

### Konvolusi Linear vs Sirkular

Untuk dua sinyal panjang $M$ dan $L$:
- **Linear**: output panjang $M + L - 1$
- **Sirkular (N-point)**: output panjang $N$ (dengan aliasing jika $N < M+L-1$)

### Overlap-Add dan Overlap-Save

Untuk pemrosesan sinyal panjang secara blok menggunakan FFT:
- **Overlap-Add**: potong input menjadi blok, proses masing-masing dengan FFT, jumlahkan overlap
- **Overlap-Save**: simpan bagian akhir blok sebelumnya untuk eliminasi circular wrap-around

---

## 2.8 DFT 2D (untuk Pengolahan Citra)

$$X[k, l] = \sum_{m=0}^{M-1} \sum_{n=0}^{N-1} x[m, n] e^{-j2\pi(\frac{km}{M} + \frac{ln}{N})}$$

Digunakan dalam kompresi citra (JPEG), filtering citra, dan analisis pola.

---

## Ringkasan

| Transform | Input | Output Frekuensi | Kegunaan |
|-----------|-------|-----------------|----------|
| **DTFT** | Diskrit, tak terbatas | Kontinu ($-\pi$ s/d $\pi$) | Analisis teori |
| **DFT** | Diskrit, panjang $N$ | Diskrit, $N$ titik | Komputasi |
| **FFT** | Diskrit, panjang $N$ (power of 2) | Diskrit, $N$ titik | Komputasi efisien |
| **STFT** | Diskrit, non-stasioner | 2D: waktu × frekuensi | Audio, biosinyal |
