# 01 — Dasar-Dasar Sinyal dan Sistem Digital

## 1.1 Apa Itu Sinyal?

Sinyal adalah representasi fisik dari informasi yang bervariasi terhadap waktu, ruang, atau variabel lainnya. Dalam DSP, kita bekerja dengan dua jenis utama:

### Sinyal Kontinu (Analog)
Sinyal kontinu $x(t)$ terdefinisi untuk setiap nilai waktu $t \in \mathbb{R}$. Contoh: gelombang suara di udara, sinyal listrik dari mikrofon.

### Sinyal Diskrit (Digital)
Sinyal diskrit $x[n]$ hanya terdefinisi pada nilai integer $n$. Diperoleh melalui proses **sampling** sinyal analog.

$$x[n] = x_a(nT_s)$$

di mana $T_s$ adalah periode sampling.

---

## 1.2 Klasifikasi Sinyal

### Berdasarkan Sifat Matematika

| Jenis | Definisi | Contoh |
|-------|----------|--------|
| **Periodik** | $x[n] = x[n+N]$ untuk semua $n$ | Gelombang sinus |
| **Aperiodik** | Tidak periodik | Sinyal suara ucapan |
| **Deterministik** | Dapat diprediksi secara matematis | $x[n] = \sin(\omega_0 n)$ |
| **Stokastik/Random** | Tidak dapat diprediksi | Noise |
| **Kausal** | $x[n] = 0$ untuk $n < 0$ | Kebanyakan sinyal real |
| **Anti-kausal** | $x[n] = 0$ untuk $n \geq 0$ | - |
| **Ganjil** | $x[-n] = -x[n]$ | Gelombang sinus |
| **Genap** | $x[-n] = x[n]$ | Gelombang kosinus |

### Energi dan Daya Sinyal

**Energi total** sebuah sinyal diskrit:
$$E = \sum_{n=-\infty}^{\infty} |x[n]|^2$$

**Daya rata-rata**:
$$P = \lim_{N \to \infty} \frac{1}{2N+1} \sum_{n=-N}^{N} |x[n]|^2$$

- Sinyal **energi terbatas** (Energy Signal): $E < \infty$, maka $P = 0$
- Sinyal **daya terbatas** (Power Signal): $0 < P < \infty$, maka $E = \infty$

## 1.3 Sinyal Dasar Penting

### 1.3.1 Impuls Satuan (Unit Impulse / Delta Dirac Diskrit)

$$\delta[n] = \begin{cases} 1 & n = 0 \\ 0 & n \neq 0 \end{cases}$$

**Sifting property**: $x[n] = \sum_{k=-\infty}^{\infty} x[k] \cdot \delta[n-k]$

Ini sangat penting: **setiap sinyal diskrit dapat dinyatakan sebagai superposisi impuls yang digeser dan diskalakan**.

### 1.3.2 Step Satuan (Unit Step)

$$u[n] = \begin{cases} 1 & n \geq 0 \\ 0 & n < 0 \end{cases}$$

Hubungan dengan impuls: $\delta[n] = u[n] - u[n-1]$ dan $u[n] = \sum_{k=0}^{\infty} \delta[n-k]$

### 1.3.3 Ramp Satuan

$$r[n] = n \cdot u[n]$$

### 1.3.4 Sinusoid Diskrit

$$x[n] = A \cos(\omega_0 n + \phi)$$

di mana:
- $A$ = amplitudo
- $\omega_0$ = frekuensi angular (rad/sample), $0 \leq \omega_0 \leq \pi$
- $\phi$ = fase awal

**Perbedaan penting dengan sinusoid kontinu**: sinusoid diskrit **periodik** hanya jika $\omega_0 / (2\pi)$ adalah bilangan rasional.

### 1.3.5 Exponensial Kompleks

$$x[n] = e^{j\omega_0 n} = \cos(\omega_0 n) + j\sin(\omega_0 n)$$

Ini adalah "eigensignal" dari sistem LTI (Linear Time-Invariant).

### 1.3.6 Deret Geometri Eksponensial

$$x[n] = a^n \cdot u[n]$$

- Jika $|a| < 1$: meluruh (stable)
- Jika $|a| > 1$: meledak (unstable)
- Jika $|a| = 1$: tetap (marginally stable)

---

## 1.4 Sistem Diskrit

Sistem adalah entitas yang **memproses sinyal input** untuk menghasilkan **sinyal output**:

$$y[n] = \mathcal{T}\{x[n]\}$$

### 1.4.1 Sifat-Sifat Sistem

#### Linearitas
Sistem $\mathcal{T}$ bersifat **linear** jika:
$$\mathcal{T}\{a_1 x_1[n] + a_2 x_2[n]\} = a_1 \mathcal{T}\{x_1[n]\} + a_2 \mathcal{T}\{x_2[n]\}$$

Mencakup dua sub-sifat:
- **Homogenitas**: $\mathcal{T}\{a \cdot x[n]\} = a \cdot \mathcal{T}\{x[n]\}$
- **Additivitas**: $\mathcal{T}\{x_1[n] + x_2[n]\} = \mathcal{T}\{x_1[n]\} + \mathcal{T}\{x_2[n]\}$

#### Time-Invariance (Kekekalan Waktu)
Jika $y[n] = \mathcal{T}\{x[n]\}$, maka:
$$\mathcal{T}\{x[n-k]\} = y[n-k]$$

Sistem yang memenuhi keduanya disebut **Linear Time-Invariant (LTI)**.

#### Kausalitas
$$y[n] \text{ tidak bergantung pada } x[n+k] \text{ untuk } k > 0$$

Output hanya bergantung pada input saat ini dan masa lalu.

#### Stabilitas (BIBO — Bounded Input Bounded Output)
Jika input terbatas maka output juga terbatas:
$$|x[n]| \leq M_x < \infty \Rightarrow |y[n]| \leq M_y < \infty$$

Syarat stabilitas untuk sistem LTI: $\sum_{k=-\infty}^{\infty} |h[k]| < \infty$

#### Memori (Memory)
- **Memoryless**: $y[n]$ hanya bergantung pada $x[n]$ saat ini
- **With memory**: $y[n]$ bergantung pada nilai $x$ masa lalu atau masa depan

---

## 1.5 Sistem LTI dan Respons Impuls

Sistem LTI sepenuhnya dikarakterisasi oleh **respons impulsnya** $h[n]$:

$$h[n] = \mathcal{T}\{\delta[n]\}$$

Output sembarang input diperoleh melalui **konvolusi**:

$$y[n] = x[n] * h[n] = \sum_{k=-\infty}^{\infty} x[k] \cdot h[n-k]$$

### Kondisi Stabilitas
Sistem LTI stabil ⟺ $\sum_{n=-\infty}^{\infty} |h[n]| < \infty$ (summable absolutely)

### Kondisi Kausalitas
Sistem LTI kausal ⟺ $h[n] = 0$ untuk $n < 0$

---

## 1.6 Persamaan Differens (Difference Equation)

Sistem LTI umumnya diimplementasikan sebagai persamaan differens:

$$\sum_{k=0}^{N} a_k y[n-k] = \sum_{k=0}^{M} b_k x[n-k]$$

Bentuk yang lebih umum (normalisasi $a_0 = 1$):

$$y[n] = -\sum_{k=1}^{N} a_k y[n-k] + \sum_{k=0}^{M} b_k x[n-k]$$

- Koefisien $b_k$ → **bagian moving average (MA)** atau feedforward
- Koefisien $a_k$ → **bagian autoregressive (AR)** atau feedback

---

## 1.7 Representasi Diagram Blok

Sistem DSP dapat digambarkan sebagai diagram blok yang terdiri dari:

1. **Unit delay** $z^{-1}$: menggeser sinyal satu sampel
2. **Adder**: menjumlahkan dua sinyal
3. **Multiplier**: mengalikan sinyal dengan konstanta

Contoh sistem MA orde-2: $y[n] = \frac{1}{3}(x[n] + x[n-1] + x[n-2])$

```
x[n] ──┬─────────────────────── × (1/3)
       │                              │
       └──→ [z⁻¹] ──┬──────── × (1/3)──→ [+] ──→ y[n]
                    │                      ↑
                    └──→ [z⁻¹] ──── × (1/3)
```

## Ringkasan

| Konsep | Keterangan Singkat |
|--------|-------------------|
| Sinyal diskrit | Terdefinisi hanya pada integer $n$ |
| Energi vs Daya | $E<\infty$ atau $0<P<\infty$ |
| Impuls $\delta[n]$ | Building block semua sinyal |
| Sistem LTI | Linear + Time-Invariant |
| Respons Impuls $h[n]$ | Karakterisasi penuh sistem LTI |
| Konvolusi | $y[n] = x[n] * h[n]$ |
| Stabilitas BIBO | $\sum|h[n]| < \infty$ |
